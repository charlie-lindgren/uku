# GIK339

**Kursnamn:** Dynamiska webbapplikationer  
**Course Name:** Dynamic Web Applications

**Poäng:** 7,5 hp  
**Nivå:** G1F — Grundnivå, kräver grundkurs  
**Ämne (SCB):** IF1 — Informatik  
**Giltig från:** 2023:1  

## Lärandemål

_Kunskap och förståelse_  
Efter avslutad kurs ska studenten kunna:

  - redogöra för grundläggande begrepp inom webbutveckling

_Färdighet och förmåga_  
Efter avslutad kurs ska studenten kunna:

  - använda CSS-ramverk för att skapa responsiva webbsidor

  - använda JavaScript för att dynamiskt påverka webbsidor

  - skapa back-end lösning för dynamisk hantering av data

## Innehåll

Kursen behandlar grundläggande begrepp inom webbutveckling. CSS-ramverk används för att skapa responsiva webbsidor. Vidare behandlar kursen programmering i front-end och back-end för att dynamiskt påverka webbsidors innehåll.

## English Version

### Learning Outcomes

_Knowledge and understanding_  
After completing the course, students will be able to:

  - explain basic concepts used in web development

_Skills and abilities_  
After completing the course, students will be able to:

  - use CSS framework to create responsive web pages

  - use JavaScript to dynamically affect web pages

  - create back-end solutions for the dynamic management of data

### Course Content

The course deals with basic concepts used in web development. CSS framework is used to create responsive webpages. In addition, the course covers front end and back end programming to dynamically affect webpage content.

### Assessment

- Assignments

  - Project

### Grades

The grading scale used for the final course grade is U–G.

Grades are reported as follows:

  - Assignments - 4 Credits | U–G
  - Project - 3.5 Credits | U–G

### Prerequisites
