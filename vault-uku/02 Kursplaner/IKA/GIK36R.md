# GIK36R

**Kursnamn:** Applikationsutveckling och testning  
**Course Name:** Application Development and Testing

**Poäng:** 7,5 hp  
**Nivå:** G1F — Grundnivå, kräver grundkurs  
**Ämne (SCB):** IF1 — Informatik  
**Giltig från:** 2023:2  

## Lärandemål

Efter godkänd kurs ska studenten kunna:

**Kunskap och förståelse**

- redogöra för centrala koncept och tekniker för mjukvarutesting.

**Färdighet och förmåga**

- skapa en testplan, utveckla testfall och testdata för olika testscenarier

- utför tester med hjälp av testverktyg och tolka resultaten

- analysera och rapportera testresultat.

## Innehåll

Denna kurs ger en introduktion till grunderna för mjukvarutestning. Centrala begrepp, metoder och aktiviteter inom mjukvarutesting gås igenom. Vidare behandlas testplanering, testdesign, testutförande och testrapportering.

## English Version

### Learning Outcomes

Upon completion of the course, students will be able to:

Knowledge and understanding

  - Describe fundamental concepts within and techniques for software testing.

Skills and abilities

  - Create a test plan, and develop test cases and test data for different test scenarios.

  - Run tests using test tools and interpret the results.

  - Analyse and report test results.

### Course Content

Course Content  
This course provides an introduction to the fundamentals of software testing. Students will learn about fundamental concepts , methods and activities relating to software testing. Topics covered include test planning, test design, test execution and test reporting.
