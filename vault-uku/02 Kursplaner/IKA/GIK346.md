# GIK346

**Kursnamn:** Data Science programmering och dataanalys  
**Course Name:** Data Science Programming and Data Analytics

**Poäng:** 15 hp  
**Nivå:** G1F — Grundnivå, kräver grundkurs  
**Ämne (SCB):** IF1 — Informatik  
**Giltig från:** 2023:1  

## Lärandemål

**Kunskap och förståelse**
Efter godkänd kurs ska studenten kunna:

- redogöra för grundläggande begrepp, metoder, algoritmer och tekniker inom data science, mjukvaruutveckling och dataanalys

**Färdighet och förmåga**
Efter godkänd kurs ska studenten kunna:

- nyttja problemlösningsstrategier i samband med kodning av mjukvara

- tillämpa algoritmer för maskininlärning

- förklara, sammanfatta och kommunicera valda lösningsmetoder och erhållna resultat

- använda metoder för att förbättra datakvalitet genom att undersöka data med avseende på inkonsekvens, felaktighet, ofullständighet eller orimliga datavärden.

- tillämpa tekniker för att omvandla data till lämpligt format för prediktering och analys

- tillämpa grundläggande statistik och sannolikhetsteori, inklusive sannolikhetsfördelningar, signifikanstester vid hypotesprovning och regression.

- implementera kontrollstrukturer, funktioner och klasser samt använda datastrukturer och filer för att lagra och manipulera data

## Innehåll

I kursen behandlas _Data Science_, dvs ett tvärvetenskapligt tillvägagångssätt för att extrahera data och upptäcka mönster genom en sammanslagning av analysmetoder, domänkompetens och teknik. I detta sammanhang behandlas hur programvara för data science utvecklas varefter studenten övas i detta.

Inom ramen för data science behandlas dataanalysdimensionerna: beskrivande, diagnostisk, prediktiv respektive föreskrivande dataanalys, som besvarar frågorna “Vad hände?“, “Varför hände det?“, “Vad kan hända i framtiden?“ respektive “Vad ska vi göra härnäst?“. Vidare behandlas maskininlärning för att modellera och lära av data samt tillämpning av denna kunskap inom ramen för automatiseringen av processer.

## English Version

### Learning Outcomes

_Knowledge and understanding  
_Upon completion of the course, students will be able to:

  - explain basic concepts, methods, algorithms and techniques in data science, software development and data analysis

_Skills and abilities_  
Upon completion of the course, students will be able to:

  - use problem-solving strategies in the context of software coding

  - apply machine learning algorithms

  - explain, summarise and communicate chosen solution methods and obtained results

  - use methods to improve data quality by examining data for inconsistency, inaccuracy, incompleteness or unreasonable data values.

  - apply methods to transform data into a suitable format for prediction and analysis

  - apply basic statistics and probability theory, including probability distributions, significance tests in hypothesis testing and regression.

  - implement control structures (sequence, selection and iteration), functions/methods and classes and use data structures and files to store and manipulate data

### Course Content

The course is about Data Science: that is to say, an interdisciplinary approach to finding, extracting and discovering patterns in data through an amalgamation of analysis methods, domain expertise and technology. In this context, how software for data science is developed is discussed, after which students receive training in this.

Within the framework of Data Science, the data analysis dimensions are dealt with: namely, descriptive, diagnostic, predictive and prescriptive data analysis, which answer the following questions in this order: What happened?, Why did it happen?, What can happen in the future?, and What should we do next?. Furthermore, machine learning to model and learn from data and the application of this knowledge within the framework of the automation of processes is covered.
