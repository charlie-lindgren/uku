# GIK2PG

**Kursnamn:** Projektbaserad systemutveckling
**Course Name:** Project-Based System Development

## Lärandemål

Det övergripande målet med kursen är att studenter i grupp i ett systemutvecklingsprojekt planerar, analyserar, designar, 
implementerar och testar utvecklat IT-system.

Efter avslutad kurs ska studenten kunna:

  - planera, analysera, designa, implementera och testa IT-system under utveckling.
  - välja och anpassa en systemutvecklingsmodell för detta IT-system.
  - välja och använda programmeringsramverk, tredje parts API:er och tjänster vid implementering av detta IT-system.
  - dokumentera och presentera framtaget IT-system.

## Innehåll

Kursen behandlar ett systemutvecklingsprojekt där planering, analys, design, implementering och test av IT-system under utveckling ingår. Grupparbete genomförs i projektform som tar sin utgångspunkt i och byggs upp från en systemutvecklingsmodell. Vidare ingår val och användning av lämplig systemutvecklingsmetod, programmeringsramverk och tredje parts API och tjänst vid detta framtagande.

## English Version

### Learning Outcomes

The main goal of the course is that students work in groups on a system development project in which they plan, analyse, design, implement, and test a developed IT system.

After completing the course, students will be able to:

  - plan, analyse, design, implement and test an IT system during development.

  - select and adapt a system development model for this IT system

  - select and use programming frameworks, and third-party API‘s and services when implementing this IT system.

  - document and present a developed IT systems.

### Course Content

The course involves a system development project that includes the planning, analysis, design, implementation and testing of an IT system that is under development. Group work is carried out in project form that is based on and built upon a system development model. The course also includes the selection and use of appropriate system development methods, programming frameworks, and third party APIs  and services during this development.
