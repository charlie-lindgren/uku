# GMI35R

**Kursnamn:** Operativsystem  
**Course Name:** Operating Systems

**Poäng:** 7,5 hp  
**Nivå:** G1N — Grundnivå, inga förkunskapskrav  
**Ämne (SCB):** DT1 — Datateknik  
**Giltig från:** 2023:2  

**Program:** DITMG  

## Lärandemål

**Kunskap och förståelse**

Efter godkänd kurs ska studenten kunna:

- redogöra för centrala begrepp, koncept, beståndsdelar och metoder inom operativsystem,

- beskriva de olika modulerna i ett operativsystem,

- exemplifiera och redogöra för hur kärnan i ett operativsystem är utformad och fungerar,

- redogöra för hur jämlöpande kod i ett operativsystem hanteras,

- redogöra för minneshantering i operativsystem.

**Färdighet och förmåga**

Efter godkänd kurs ska studenten kunna:

- installera, konfigurera samt administrera olika operativsystem avsedda för klientmiljöer,

- handhava och administrera olika operativsystems filsystem, användare och rättigheter.

## Innehåll

Kursen ger en introduktion till hur operativsystem är uppbyggda och fungerar. Både teoretiska och praktiska aspekter av operativsystem kommer att studeras. I teorimomentet behandlas grundläggande principer för operativsystem. Detta moment innefattar exempelvis mjukvarurelaterade objekt såsom adressering, processer och trådar, resurs- och minneshantering såsom virtuellt minne, filsystem och filhantering, samtidighet/parallellitet, synkronisering och schemaläggning. Även virtualisering och moln/molntjänster berörs grundläggande. I det praktiska momentet behandlas installation, konfigurering, användaradministration och rättighetssystem. Dessutom introduceras grunderna i skriptspråk och hur de kan användas för systemadministration i olika operativsystemsmiljöer.

## English Version

### Learning Outcomes

**Knowledge and Understanding**

Upon completion of the course, the student should be able to:

  - Explain central concepts, principles, components, and methods within operating systems.

  - Describe the different modules in an operating system.

  - Provide examples and explain how the kernel of an operating system is designed and how it functions.

  - Explain how concurrent code is handled in an operating system.

  - Explain how memory is managed in an operating system.

  
**Skills and Abilities**

Upon completion of the course, the student should be able to:

  - Install, configure, and administer various operating systems intended for client environments.

  - Manage and administrate different operating systems‘ file systems, users, and permissions.
