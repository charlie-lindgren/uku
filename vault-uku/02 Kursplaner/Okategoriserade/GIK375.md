# GIK375

**Kursnamn:** Komponentdriven webbdesign  
**Course Name:** Component Driven Web Design

**Poäng:** 7,5 hp  
**Nivå:** G1F — Grundnivå, kräver grundkurs  
**Ämne (SCB):** IF1 — Informatik  
**Giltig från:** 2023:2  

**Program:** DSVPG  

## Lärandemål

Efter godkänd kurs ska studenten kunna:

_Färdighet och förmåga_

  - utveckla modulära och återanvändbara webbkomponenter genom front-end,

  - manipulera DOM för att uppnå avsedd struktur och funktionalitet,

  - använda CSS-ramverk för att uppfylla designkrav,

_Värderingsförmåga och förhållningssätt_

  - reflektera över hållbar webbdesign.

## Innehåll

Kursen behandlar tekniker inom DOM-manipulation av webbsidors struktur och funktionalitet. Komponentdriven webbdesign behandlas som helhet i ett projektarbete under kursens gång. I detta projekt tillämpas ramverk för att uppfylla designkrav.

I kursen sker även reflektion kring hållbar webbdesign och dess implikationer.

## English Version

### Learning Outcomes

Upon completion of the course, students will be able to:

_Skills and abilities_

  - develop modular and reusable web components through front-end,

  - manipulate the DOM to achieve the desired structure and functionality,

  - utilise CSS frameworks to meet design requirements,

_Judgement and approach_

  - reflect on sustainable web design.

### Course Content

The course covers techniques in DOM manipulation for the structure and functionality of web pages. Furthermore, the application of CSS frameworks to fulfill design requirements is also explored. During the course, students consolidate their acquired knowledge through a project focusing on component-driven web design.

During the course, reflection on sustainable web design and its implications also occurs.
